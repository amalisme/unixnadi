Why I Built This

I kept seeing the same problem: small business owners in the beauty and wellness space either paying too much for overbuilt WordPress themes or struggling with page builders that slow their sites to a crawl. They don't need a CMS for a 5-page website. They need something fast, professional, and easy to maintain.

Bloom is my answer to that. It's a template that loads instantly, looks great on every device, and doesn't require you to learn a framework or install Node.js just to change a color.

Key Features

Here's what makes Bloom different from the typical template you'll find on ThemeForest:

Zero Dependencies. No jQuery, no Bootstrap, no frameworks. Just clean, vanilla code that you can read and understand. This means fewer things to break and faster load times.

Built-in Dark Mode. A toggle that actually works, with localStorage persistence so your visitors' preference is remembered. It's a small touch that makes a big difference.

Fully Responsive. Mobile-first design that looks stunning on phones, tablets, and desktops. I've tested it across devices because nothing kills credibility faster than a broken mobile layout.

Performance Optimized. Bloom is built with Core Web Vitals in mind. Fast Largest Contentful Paint, minimal Cumulative Layout Shift, and snappy interactions. Google will thank you.

Accessible. WCAG AA compliant with semantic HTML, proper ARIA labels, full keyboard navigation, and visible focus states. Everyone should be able to use your website.

Easy to Customize. Colors and fonts are controlled through CSS variables. Want to change the primary color? Update one line. The code is well-commented and organized so you can find what you need quickly.

Technical Details

For those who care about what's under the hood:

  • Pure HTML5, CSS3, and ES6+ JavaScript
  • CSS Custom Properties for easy theming
  • Flexbox and CSS Grid layouts
  • CSS-only animations (scroll reveal, hover effects)
  • Mobile navigation with full accessibility support
  • Contact form ready for Formspree, Netlify Forms, or your custom backend
  • SEO-friendly semantic markup throughout
  • Print stylesheet included
  • Google Fonts (Playfair Display + Inter)
